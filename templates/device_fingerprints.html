// How Device Fingerprinting Works
A lightweight JavaScript snippet runs silently in the user's browser and collects 8 device signals: screen resolution, timezone, canvas hash, WebGL vendor, plugins, fonts, language, and webdriver flag. Each session is analysed for 10 fraud signals including headless browsers, VPN ASNs, timezone mismatches, and anti-detect browser patterns. The fingerprint collector is automatically embedded in all DefenceIQ pages for logged-in users.
